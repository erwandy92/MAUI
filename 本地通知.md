我先查一下最新版本和設定方式。以下是完整的設定步驟，基於最新版 13.0.0（2025/11/30 更新）：



\---



\## Step 1 — 安裝套件



```bash

dotnet add package Plugin.LocalNotification

```



\---



\## Step 2 — Android 權限設定



在 `Platforms/Android/AndroidManifest.xml` 加入：



```xml

<uses-permission android:name="android.permission.VIBRATE" />

<uses-permission android:name="android.permission.POST\_NOTIFICATIONS" />

<uses-permission android:name="android.permission.WAKE\_LOCK" />

<uses-permission android:name="android.permission.RECEIVE\_BOOT\_COMPLETED" />



<!-- 排程精確時間（選用） -->

<uses-permission android:name="android.permission.SCHEDULE\_EXACT\_ALARM" android:maxSdkVersion="32" />

```



\---



\## Step 3 — iOS 設定



在 `Platforms/iOS/Info.plist` 加入：



```xml

<key>UIBackgroundModes</key>

<array>

&#x20;   <string>fetch</string>

&#x20;   <string>remote-notification</string>

</array>

```

\## Step 4 — MauiProgram.cs 初始化



```csharp

using Plugin.LocalNotification;



public static class MauiProgram

{

&#x20;   public static MauiApp CreateMauiApp()

&#x20;   {

&#x20;       var builder = MauiApp.CreateBuilder();

&#x20;       builder

&#x20;           .UseMauiApp<App>()

&#x20;           .UseLocalNotification(config =>

&#x20;           {

&#x20;               // Android 通知頻道設定

&#x20;               config.AddAndroid(android =>

&#x20;               {

&#x20;                   android.AddChannel(new NotificationChannelRequest

&#x20;                   {

&#x20;                       Id = "general",

&#x20;                       Name = "General",

&#x20;                       Description = "一般通知"

&#x20;                   });

&#x20;               });

&#x20;           });



&#x20;       return builder.Build();

&#x20;   }

}

```



\---



\## Step 5 — 請求通知權限



在 `App.xaml.cs` 或頁面初始化時：



```csharp

protected override async void OnStart()

{

&#x20;   if (await LocalNotificationCenter.Current.AreNotificationsEnabled() == false)

&#x20;   {

&#x20;       await LocalNotificationCenter.Current.RequestNotificationPermission();

&#x20;   }

}

```



\---



\## Step 6 — 發送通知



\### 立即發送

```csharp

var notification = new NotificationRequest

{

&#x20;   NotificationId = 1,

&#x20;   Title = "Hello!",

&#x20;   Description = "這是一則本地通知",

};



await LocalNotificationCenter.Current.Show(notification);

```



\### 排程發送（5 分鐘後）

```csharp

var notification = new NotificationRequest

{

&#x20;   NotificationId = 100,

&#x20;   Title = "提醒",

&#x20;   Description = "5 分鐘後的通知",

&#x20;   Schedule = new NotificationRequestSchedule

&#x20;   {

&#x20;       NotifyTime = DateTime.Now.AddMinutes(5)

&#x20;   }

};



await LocalNotificationCenter.Current.Show(notification);

```



\### 重複通知（每週）

```csharp

var notification = new NotificationRequest

{

&#x20;   NotificationId = 200,

&#x20;   Title = "每週提醒",

&#x20;   Description = "每週固定通知",

&#x20;   ReturningData = "some\_data",

&#x20;   Schedule = new NotificationRequestSchedule

&#x20;   {

&#x20;       NotifyTime = DateTime.Now.AddDays(7),

&#x20;       NotifyRepeatInterval = TimeSpan.FromDays(7)

&#x20;   }

};



await LocalNotificationCenter.Current.Show(notification);

```

\## Step 7 — 處理通知點擊事件（選用）



```csharp

LocalNotificationCenter.Current.NotificationActionTapped += OnNotificationTapped;



private void OnNotificationTapped(NotificationActionEventArgs e)

{

&#x20;   // e.Request.ReturningData 可取得自訂資料

&#x20;   Debug.WriteLine($"通知被點擊，資料：{e.Request.ReturningData}");

}

```



\---



\## 常用 API 整理



| 方法 | 說明 |

|------|------|

| `Show(request)` | 顯示/排程通知 |

| `Cancel(id)` | 取消特定通知 |

| `CancelAll()` | 取消所有通知 |

| `AreNotificationsEnabled()` | 檢查是否已開啟通知 |

| `RequestNotificationPermission()` | 請求通知權限 |
